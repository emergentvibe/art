# Emergent Art Workshops Website

A static website for Emergent Art Workshops, featuring a playful purple, green, pink, and white theme with a typewriter-like font.

## Files

- `index.html` - The main website page
- `styles.css` - CSS styling with the specified theme
- Image files - Workshop images used in the website
- `README.md` - This documentation file

## Deployment Instructions (GitHub Pages)

1. Create a new GitHub repository:
   - Sign in to your GitHub account
   - Click the "+" icon in the top right and select "New repository"
   - Name your repository (e.g., `emergent-art-workshops`)
   - Set it to public
   - Click "Create repository"

2. Upload your files:
   - You can drag and drop all the files from this directory into the GitHub repository, or use git commands to push them

3. Enable GitHub Pages:
   - Go to your repository's "Settings" tab
   - Scroll down to the "GitHub Pages" section
   - Under "Source", select "main" (or "master") branch
   - Click "Save"

4. Access your website:
   - After GitHub Pages is enabled, you'll see a message with your website URL
   - It will be in the format: `https://[your-username].github.io/[repository-name]/`
   - The site may take a few minutes to become available

## Customization

If you want to make changes to the site:

- Edit `index.html` to update content
- Edit `styles.css` to change the styling
- Replace images as needed
- After pushing changes to GitHub, the site will automatically update

## Local Development

To view the site locally before deploying:
1. Download all files to your computer
2. Open the `index.html` file in your web browser
